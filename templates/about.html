This site sends you emails with recommendations of new papers based on the tags you add to papers.

For get better recommendations, add more tags. To find new tags, you can discover papers through the "similar" button.

  1. Log in
  2. Browse the corpus of papers [1] and tag some papers
  3. Get email recommendations

How it works: We periodically poll the arxiv API for new papers. You can tag papers of interest, then recommends new papers for each tag based on SVMs over tfidf features of paper abstracts. You can search, rank and sort, slice and dice these results in a pretty web UI. Lastly, arxiv-sanity-lite can send you daily emails with recommendations of new papers based on your tags. Curate your tags, track recent papers in your area, and don't miss out!

[1] This instance has a corpus of about 50,000 papers from the following categories of arxiv:

  • cs.HC - Human-Computer Interaction
  • cs.CY - Computers and Society
  • cs.CV - Computer Vision and Pattern Recognition
  • cs.MM - Multimedia
  • cs.OH - Other Computer Science
  • cs.SD - Sound
  • cs.GR - Graphics
  • cs.DL - Digital Libraries

To request other categories or sources, contact the admin.

The code for this website is open source at Github under arxiv-sanity-lite.

Developed in spare time as a labor of love by @karpathy.
